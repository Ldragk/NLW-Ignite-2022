<h1>Instalação</h1><br><br>

</div>

<p>$ cd server</p>
<p>$ type nul > .env</p>


<br>

<p>Dentro do arquivo .env</p><br>

DATABASE_URL="file:../src/database/db.sqlite"<br><br><br>

<h2>Servidor</h2><br>

<p>$ cd server</p> 
<p>$ npm install</p>
<p>$ npx prisma generate</p>
<p>$ npm run dev</p>
<br>

<h2>Web</h2><br>

<p>$ cd web</p>
<p>$ npm install</p>
<p>$ npm run dev</p>
