<h1>Instalação</h1><br><br>

</div>

<h2>Criar um arquivo .env na pasta do servidor</h2><br>

<p>$ cd server</p>
<p>$ type nul > .env</p>


<br>

<h3>Dentro do arquivo .env</h3><br>

DATABASE_URL="file:../src/database/db.sqlite"<br><br><br>

<h2>Servidor</h2><br>

<p>$ cd server</p> 
<p>$ npm install</p>
<p>$ npx prisma generate</p>
<p>$ npm run dev</p>
<br>

<h2>Web</h2><br>

<p>$ cd web</p>
<p>$ npm install</p>
<p>$ npm run dev</p>
